# Changelog

All notable changes to the IBEW Theme will be documented in this file.

## [1.0.0] - 2025-12-01

### Added
- Initial release of IBEW Theme for Drupal 11
- Based on Radix 6.x with Bootstrap 5
- Custom IBEW color scheme (Dark Blue #003366 and Gold #ffd700)
- Responsive layout with Bootstrap grid system
- Component-based architecture using Drupal's Single Directory Components
- IBEW Header component with logo and navigation
- Modern build system using Laravel Mix
- Sass compilation with Bootstrap variable overrides
- ES6 JavaScript support
- BrowserSync integration for development
- Comprehensive documentation (README, QUICKSTART, STRUCTURE)
- Basic page template with sidebar support
- Theme hooks for customization
- Responsive breakpoints matching Bootstrap 5
- SVG logo placeholder
- .gitignore for build artifacts

### Features
- **Regions**: Header, Content, Sidebar First, Sidebar Second, Footer
- **Components**: Reusable IBEW Header component
- **Build Tools**: npm scripts for dev, watch, and production builds
- **Styling**: Custom SCSS with Bootstrap 5 integration
- **Documentation**: Complete setup and development guides

### Technical Details
- Drupal core compatibility: ^10 || ^11
- PHP requirement: 8.3+
- Node.js requirement: 16+
- Base theme: Radix
- CSS Framework: Bootstrap 5.3.2
- Build tool: Laravel Mix 6.0.49
